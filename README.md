# Spring Boot 101 (스프링 부트 101)
- 스프링 부트를 진짜 진득하게 파보자
- /src 경로로 이동해서 보면 됨
- 소크라테스의 정신으로 진짜 물어뜯어 보자
- 왜 하필 스프링 부트냐? 뭐 지들이 나름 정해둔게 많으니까 그만큼 공부할게 많아서
- 계속 업데이트 됨

### 대상
- 자바와 스프링 부트를 처음 시작하지만 다른 프레임워크나 언어 개발 경험이 있는 개발자
- 스프링 부트를 그냥 실무에서 바로 만나가지고 따로 공부할 시간 없이 그냥 유지보수만 하다보니 답답한 개발자
- 솔직히 아예 개발 모르는 사람이 보기에는 어려울 수 있음
