# Spring Boot 101 (스프링 부트 101)
- 스프링 부트를 진짜 진득하게 파보자
- /src 경로로 이동해서 보면 됨
- 소크라테스의 정신으로 진짜 물어뜯어 보자
- 왜 하필 스프링 부트냐? 뭐 지들이 나름 정해둔게 많으니까 그만큼 공부할게 많아서
- 계속 업데이트 됨

### 대상
- 자바와 스프링 부트를 처음 시작하지만 다른 프레임워크나 언어 개발 경험이 있는 개발자
- 스프링 부트를 그냥 실무에서 바로 만나가지고 따로 공부할 시간 없이 그냥 유지보수만 하다보니 답답한 개발자
- 솔직히 아예 개발 모르는 사람이 보기에는 어려울 수 있음

### 목차
- 1.0 버전 완성되면 업데이트

### 현재버전
- Draft

### ETC
- Draft 버전으로 시작 됨
- Draft 버전은 따로 버전 표시 없고 수시로 커밋되어서 수정 됨
- 전체 문서가 일단 구조가 잡히면 1.0 버전으로 올림
- 이후 단순 부분적 수정은 1.0.X 버전으로 내부적으로는 생각하는데 따로 표기는 따로 안함
- 메이저한 내용이나 신규 내용 추가시에는 1.1 , 1.2 등 버전 업데이트
- 완성된 개별 문서가 대대적으로 수정 되거나 순서와 내용이 바뀌면 2.0, 3.0등 메이저 버전 올림 (거의 없을듯)
